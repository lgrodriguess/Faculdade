# 📚 Faculdade

Este repositório reúne **exercícios, trabalhos e provas** das disciplinas de programação que estou cursando durante a minha graduação/curso.

O objetivo é **organizar meu aprendizado**, acompanhar minha evolução e manter um histórico dos conteúdos estudados.

---

## 📂 Estrutura do Repositório

O repositório está organizado por **disciplinas** ou **matérias**, e dentro de cada pasta estão os exercícios, trabalhos e avaliações correspondentes.
